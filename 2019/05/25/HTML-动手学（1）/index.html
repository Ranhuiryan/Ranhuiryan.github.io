何谓 HTML

HTML (HyperText Markup Language):

  • Markup 语言为定义纯文本结构以及版式的编程语言。
  • 计算机可以解读 HTML 元素中的纯文本。
  • HyperText（超文本）为可链接到其他文本的文字，也被称为 hyperlinks（超链接）。

学习 HTML 是建立个人网站的第一步，本次仅涉及添加和修改网页上的基本内容，包括文本、图片以及视频。

HTML 注释

HTML 是由各种元素构成的，这些元素构成了网页以及定义了其中的内容。

下图为 HTML 段落元素的结构，它由一个 opening tag <p>，文本内容 “Hello World!”，以及一个 closing tag </p> 组成。一个 tag 及其里面的内容即称为 HTML 元素。除此之外，还有其他用于组织和展示文本以及其他内容（如图片）的标识符。

上图中组成元素的各个部分：

  • HTML 元素（简称“元素”）—— HTML 文本中内容的单位，由 HTML 标识符以及其中的内容组成。
  • HTML 标识符 —— 元素名称，用箭头括号 <> 括起来的部分。
  • 开始标识符 —— 第一个 HTML 标识符，标志一个 HTML 元素的开始，用箭头括号 <> 括起来。
  • 内容 —— 夹在开始与结束标识符中的信息（文本或其他元素）。
  • 结束标识符 —— 第二个 HTML 标识符，标志一个 HTML 元素的结束。结束标识符在左箭头括号后面要跟随一个斜线 /。

HTML 结构

HTML 由家族树关系构成。当一个元素包含到另一个元素中时，前者称为后者的子元素。子元素被称为嵌套在父元素中。

在上述例子中，<p> 元素嵌套在 <body> 元素中。<p> 元素是 <body> 元素的子元素，<body> 元素是 <p> 元素的父元素。为增加代码的可读性，可以加2格空格缩进。

由于可以有多层嵌套，因此可以衍生出次子元素、次次子元素等等。祖先元素和子元素的关系称为层次结构（hierarchy）。
