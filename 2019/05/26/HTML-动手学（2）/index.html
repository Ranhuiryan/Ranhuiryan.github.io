Divs

<div> 为 HTML 中的常用元素之一。<div> 为单词“division”的简写，也可称为将页面分为不同章节的容器。

<div> 中可包含各种文本以及 HTML 元素，例如链接、图片或者视频。为保证可读性，在 <div> 中嵌套元素时请务必在每行添加两空格缩进。

Attributes（属性）

使用属性展开元素标签。属性是加在元素开始标识符中的一段文字，可以用于说明元素的信息以及变换格式。属性有以下2部分组成：

  • 属性名称
  • 属性值

id 为常用的一个属性。如果要多次使用同一种元素的话（如 <div>），可以使用该属性区分该元素中包含的内容。除此之外，id 还具备其他功能。
