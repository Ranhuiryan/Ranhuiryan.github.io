Text Input（文字输入）

使用 <input> 元素在表单中创建输入域。

<input> 的 type 属性值决定了输入域在网页中的渲染方式以及接受的数据类型。

"text" 值将在网页中渲染一个文字输入框，即type="text"。此时还需加入 name 属性，否则输入的信息将不会被上传到网页。

例如，当用户在此输入框中输入 “important details”

当提交表单时，"first-text-field=important details" 会被上传至网页 /example.html。

也可以定义 value 属性的值以向输入框中添加默认的输入信息。
