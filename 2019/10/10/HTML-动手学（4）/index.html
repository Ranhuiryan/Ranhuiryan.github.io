HTML 动手学（4）

发表于 2019-10-10 更新于 2020-10-09

Codecademy HTML 教程学习笔记

Images（图片）

使用 <img> 标识符向网页中添加图片。

注意，<img> 是自闭合标识符，在 <img> 标识符后包含一条斜线 /。自闭合标识符后面的 / 可以忽略。

<img> 有一个必要的属性 src，src 即为图片源或图片的位置。此例中，src 的值应为图片的统一资源定位器（URL）地址。

Images Alts（图片替代文字）

<img> 的另一个属性 alt 可以为图片添加一段说明文字，其值便是说明文字的内容：

alt 还有以下几个作用：

  • 图片加载失败时显示替代文字；
  • 一些软件可以为视力障碍读者朗诵图片替代文字的内容；
  • 便于搜索引擎搜索。

alt 属性并非必须。

Video（视频）

使用 <video> 标识符向网页中添加视频。

<video> 有一个必要的属性 src，src 即为视频源或视频的位置。<video> 元素包含一个 <video> 开始符和一个 <video> 结束符。

在 src 属性之后，还包含 width 和 height 属性分别用于定义视频的长和宽。controls 属性表示添加必要的视频控件，如：暂停、播放和跳过等。

<video> 标识符中间的文字在视频无法加载时会显示在网页上。
