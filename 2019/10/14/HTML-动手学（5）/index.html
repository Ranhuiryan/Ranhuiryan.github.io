The tag（HTML 标识符）

在 <!DOCTYPE html> 后加入 <html> 标识符，以创建 HTML 结构和内容。

The Head（开头）

在 <html> 标识符后加入 <head> 标识符，向页面中添加元数据（在页面中无法直接展示的内容）

Page Titles（页标题）

在 <head> 中插入 <title> 标识符以定义页标题。

Linking to Other Web Pages（连接到其他页面）

通过添加锚元素 <a> ，创建一段文字的超链接。

定义超链接参考属性 herf 将该段文字链接到指定页面。

链接到相邻页面

如要访问同一根目录下的其他页面，如对于如下根目录：

可以添加目标页面的相对路径到 href 值。

Opening Links in a New Window（在新窗口打开链接）

通过 <a> 的 target 属性控制打开链接的行为。

若要使链接在新窗口中打开，需要将 target 的值设为 _blank。

注意：对于现代的浏览器，添加 target="_blank" 属性会在新标签页中打开页面。

Linking At Will（在各种内容中添加超链接）

可以将任意 HTML 元素添加到锚元素中。

Linking to Same Page（链接到同一页面上的位置）

首先在目标位置定义 id。

此时在目标链接中定义 # +对应 id 的值，即可链接到相应位置。

Comments（注释）

在 <!-- 与 --> 中间插入注释。

在Mozilla 文档查找更多的 HTML 标识符。
