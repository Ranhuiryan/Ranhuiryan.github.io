HTML-动手学（6）

发表于 2019-10-15 更新于 2020-10-09

Codecademy HTML 教程学习笔记

提要

TABLES（表格）

Create a Table（创建表格）

使用 <table> 元素创建表格。

Table Rows（行）

使用表格行（table row）元素 <tr> 向表格中添加行。

Table Data（表格数据）

使用表格数据（table data）元素 <td> 向表格中添加数据。

Table Headings（表索引）

使用表索引（table Headings）元素 <th> 向表格中添加表索引。

注意 <th> 的 scope 属性，该属性值可取以下两种之一：

  • row - 行索引
  • col - 列索引

Table Borders（表格边框）

在旧版的 HTML 中，可以通过在 <tabel> 中添加 border 属性来指定表格的边框。

注意：上述代码已不在被使用，与此同时，应在对应的 CSS 样式表中指定表格边框的宽度。

Spanning Columns（合并列）

使用 colspan 属性合并列，其值为要合并的列数。

Spanning Rows（合并行）

使用 rowspan 属性合并行，其值为要合并的行数。

Table Body（表体）

使用表体（table body）元素 <tbody> 创建表体。

<tbody> 元素应包含所有表格数据，但不包含表索引。

注意：此处 <tbody> 中包含有表索引，此问题会在下一节得到解决。

Table Head（表头）

使用 <thead> 元素在表格第一行创建表头。

Table Footer（页脚）

使用 <tfoot> 元素在表格最后一行创建页脚。

Styling with CSS（使用 CSS 添加样式）

使用 CSS 添加表格样式。
